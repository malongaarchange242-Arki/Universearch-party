Politique de Confidentialité

Nous respectons votre vie privée et protégeons vos données personnelles.

Quelles données collectons-nous ?

Informations personnelles

Lors de votre inscription, nous collectons :

  • Nom et Prénom - Pour identifier votre pass
  • Email - Pour vous contacter et confirmations
  • Numéro de téléphone - Pour communications urgentes
  • École/Université - Pour statistiques démographiques
  • Informations de paiement - Numéro pour MTN MoMo ou Airtel Money (stocké de manière sécurisée)

Données techniques

Nous collectons automatiquement :

  • Adresse IP
  • Type de navigateur
  • Logs d'accès
  • Cookies de session

Comment utilisons-nous vos données ?

  • Créer votre ticket - Génération du QR code et du pass événementiel
  • Traiter les paiements - Validation et suivi de vos transactions MTN/Airtel
  • Communications - Envoi de confirmations, rappels et informations importantes
  • Sécurité - Prévention des fraudes et usage abusif
  • Améliorations - Analyse de l'utilisation pour améliorer nos services
  • Statistiques anonymes - Rapports d'événements (sans identification personnelle)

Comment protégeons-nous vos données ?

  • Chiffrement - Vos données sensibles sont chiffrées en transit (HTTPS/SSL)
  • Stockage sécurisé - Les informations de paiement sont stockées de manière sécurisée
  • Accès limité - Seul le personnel autorisé peut accéder aux données
  • Audit de sécurité - Vérifications régulières de la sécurité des données

Combien de temps conservons-nous vos données ?

  • Pendant l'événement - Données conservées pour accès et entrée
  • Après l'événement - 1 an pour archivage et conformité légale
  • Suppression - Vous pouvez demander la suppression de vos données à tout moment

Vos droits

Vous avez le droit de :

  • Accéder - Demander accès à vos données personnelles
  • Corriger - Modifier vos informations inexactes
  • Supprimer - Demander la suppression de vos données (droit à l'oubli)
  • Transférer - Recevoir vos données dans un format lisible
  • Vous opposer - Refuser certains usages de vos données

Pour exercer ces droits, contactez : support@universearch.com

Questions ?

Si vous avez des questions concernant cette politique de confidentialité ou nos pratiques de données, veuillez nous contacter :

  • Email : support@universearch.com
  • WhatsApp : Disponible prochainement
